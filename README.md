# Rock_Paper_Scissors
Этот код реализует простую консольную игру «камень, ножницы, бумага», в которой пользователь играет против компьютера. Программа запрашивает у пользователя ввод, затем случайным образом выбирает ход компьютера, сравнивает оба варианта и определяет победителя или ничью. После каждого раунда пользователю предлагается сыграть ещё раз, поэтому игра может продолжаться неограниченное количество раз, пока пользователь сам не решит её завершить.
